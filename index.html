HTML and CSS

HTML (HyperText Markup Language) Tags and their meaning

ELEMENTS
Tags Names Description
<li> List Item Used to create list items inside <ol> and <ul>
<video> Video Element Embeds a media player for video playback
<em> Emphasis Element Used to italicize the text
<ol> Ordered List Element Used to create a list of items in sequential order
<ul> Unordered List Element Used to create a list of items in no particular order
<div> Div Element Used as a container that divides an HTML document into sections
<br> Line Break Element Creates a line break in text
<img> Image Element Embed images in documents
<h1> - <h6> Heading Elements Used to create headers with <h1> being the highest to the <h6> being the lowest
<p> Paragraph Element Contains and display a block of text
<strong> Strong Element Highlights important, serious, or urgent text and browsers by making it bold
<a> Anchor Element Used to create hyperlinks in an HTML document
